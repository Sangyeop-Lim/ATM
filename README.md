# 💳 ATM Unity Project

Unity로 제작된 간단한 **ATM 시뮬레이션 프로젝트**입니다. 사용자는 ATM UI를 통해 입금, 출금, 잔액 확인 등의 기능을 체험할 수 있습니다.

---

## 📁 프로젝트 구조

```plaintext
ATM-main/
├── Assets/
│   └── Scenes/
│       └── SampleScene.unity         # 메인 씬
├── Packages/
│   ├── manifest.json                 # Unity 패키지 목록
│   └── packages-lock.json            # 패키지 버전 잠금
├── ProjectSettings/
│   └── *.asset                       # 프로젝트 세팅 파일들
├── .gitignore
├── .vsconfig
